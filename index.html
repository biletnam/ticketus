Ticketus is a simple, no-frills ticketing system for helpdesks

Tired of trying to get RT or Spiceworks Help Desk to do what you want? Overwhelmed by the amount of configuration options and workflows that other ticketing systems support?

Then try Ticketus, a simple, no-frills ticketing system for helpdesks. It doesn't include all the advanced workflows that other ticketing systems have, meaning you can focus on the most important thing: tickets!

You can have it up and running in minutes, with very little configuration required.

Features

  • Modern, minimalist web interface
  • Tag tickets any way you like
  • No forced workflow
  • Automatically reference other tickets/comments (coming soon!)
  • Create and reply to tickets through an email interface (supports POP3) (coming soon!)
  • Markdown support for comments
  • Import tickets from other systems such as GitHub Issues

Software requirements

  • Python 3.3+
  • PostgreSQL 9.3+
  • WSGI server (e.g. gunicorn)
  • Web server (e.g. nginx or Apache2)

Full installation instructions are available in the README.md file on GitHub.
