Ticketus is a simple, no-frills ticketing system for helpdesks

Tired of trying to get RT or Spiceworks Help Desk to do what you want? Then try Ticketus, a simple, no-frills ticketing system for helpdesks. It doesn't include all the advanced workflows that other ticketing systems have, meaning you can focus on the most important thing: tickets!

You can have it up and running in minutes, with very little configuration required.

Features

  • Modern, minimalist web interface
  • Tag tickets any way you like
  • Reference other tickets
  • Create and reply to tickets through email interface

Software requirements

  • Python 3.3+
  • PostgreSQL 9.3+
  • WSGI server (e.g. gunicorn)
  • Web server (e.g. nginx or Apache2)
